# Customization Questions

Use these questions before launching the site.

1. What is the church address and official contact email?
2. What are the final service times?
3. Who are the pastor and ministry leaders?
4. Which donation provider should be connected?
5. Which social media links should appear in the footer?
6. Should sermon media be uploaded locally or embedded from YouTube, Vimeo, or an audio host?
7. Which admin users should be created before launch?
8. What privacy policy should be used for prayer requests and newsletter subscriptions?
